# PicklePro / Court Chic

Premium Pickleball Equipment e-commerce website.

## Project info

A modern, high-performance web application built for dedicated pickleball players.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally:

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
